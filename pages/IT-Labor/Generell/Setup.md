# Umgebungseinrichtung

## Systemvoraussetzungen

### Auf Windows

- Windows 11 oder später
- WSL 2 (Windows Subsystem for Linux 2)
- VSCode
- Git (oder über WSL)
- Docker Desktop für Windows

### Auf Linux

- Git
- VSCode
- Docker
- Java 21+
- Maven

### Auf macOS

- Homebrew
- VSCode
- Docker Desktop für Mac
- Git
- Java 21+
- Maven

## Installation Checkliste

- [ ] VSCode installiert
- [ ] Git konfiguriert (`git config --global user.name "..."`)
- [ ] Docker installiert und läuft
- [ ] Java 21 verfügbar (`java --version`)
- [ ] Maven installiert (`mvn --version`)

## Erste Schritte

1. Lies die [Würschtlcode](../Würschtlcode/WSL-und-VSCode.md) Seite für dein Betriebssystem
2. Konfiguriere [Git](../Git/Grundlagen.md) mit deinen Daten
3. Verstehe das [Git Workflow Modell](../Git/Workflow.md)
4. Lerne [Docker und Devcontainer](../Docker/Grundlagen.md)
