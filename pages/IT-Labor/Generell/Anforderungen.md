# IT-Labor: Anforderungen

## Kursrahmen

- **Leitung**: Herr Schütz
- **Fokus**: Java-Programmierung
- **Umgebung**: Linux/WSL (WSL nicht notwendig, aber unterstützt)
- **Schwerpunkt**: Professionelle Entwicklungswerkzeuge

## Lernziele

Dieser Kurs vermittelt praktische Fähigkeiten für die professionelle Softwareentwicklung:

- Versionskontrolle mit Git
- Containerisierung und Deployment mit Docker
- Remote Development mit VSCode
- Professionelles Java-Setup mit Maven
- Zusammenarbeit in Teams

## Anforderungen an dich

- Bereitschaft, neue Werkzeuge zu lernen
- Verständnis für Kommandozeile/Terminal
- Systematisches Arbeiten
- Neugier und Experimentieren (Code ist keine Einbahnstraße!)

## Besonderheiten

Die praktische Ausbildung orientiert sich an realen Entwicklungsprozessen. Das heißt:

- Professionelle Werkzeuge von Anfang an
- Nicht nur Code, sondern auch Zusammenarbeit und Versionierung
- Docker und Devcontainer für konsistente Umgebungen
- Best Practices aus der Industrie

## Materialien & Ressourcen

- VSCode als IDE
- Git für Versionskontrolle
- Docker Desktop für lokale Entwicklung
- Java 21 + Maven
